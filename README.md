## Installation

To activate individual virtual environment:

```text
$ pip install virtualenv 
```

To create and start in project root:

```text
$ virtualenv venv
$ source venv/bin/activate
```

Install dependencies in virtual environment:

```text
$ pip install -r requirements.txt
```
